# Sync Spreadsheet to web
Google Spreadsheet や Excel で管理しているリストのデータをウェブ上に表示するためのものです。

Codeigniter を土台にして作られています。

API で CSV を受け付けて、JSON に変換して保存。

View で JSON のデータを表示しています。
